README

Перед установкой откройте папку "Зайти перед установкой".

В ней находятся:
• Profile — профиль Kui Nameplates.
• WA.txt — WeakAura.
• README.txt — инструкция.

1. Установка профиля

Откройте папку:

World of Warcraft\WTF\Account\ВАШ_ЛОГИН\SavedVariables

Скопируйте содержимое папки Profile в папку SavedVariables.

2. Установка аддона

Скопируйте папки:

• Kui_Nameplates
• Kui_Nameplates_Auras

в папку:

World of Warcraft\Interface\AddOns

3. Проверка профиля

Запустите игру.

Откройте:

Esc → Интерфейс → Модификации → Kui Nameplates → Профили

Выберите профиль:

Dozawow Neverest x3

Откройте:

Esc → Интерфейс → Индикаторы здоровья → Снять галочку с "Новые индикаторы здоровья"

После выбора выполните команду:

/reload

4. Импорт WeakAura

Импортируйте WeakAura из файла WA.txt.

После этого откройте:

Esc → Интерфейс → Имена

Включите:

✔ Наложение индикаторов друг на друга
✔ Индикаторы противников
✔ Индикаторы союзников

После этого выполните:

/reload

Готово!

Теперь аддон полностью установлен и готов к работе.

При желании вы можете кастомизировать данный профиль под себя: изменить размеры, шрифты, цвета, расположение элементов и любые другие настройки в соответствии со своими предпочтениями.
